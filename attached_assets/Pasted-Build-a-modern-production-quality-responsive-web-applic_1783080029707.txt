Build a modern, production-quality responsive web application prototype in french called **EcoCollect AI**.



This is **ONLY a frontend prototype**.



**DO NOT build any backend.**

**DO NOT use a database.**

**DO NOT create APIs.**



Everything should use realistic mock data stored in JSON objects.



The application should look like a real SaaS product ready for investors or university presentation.



---



# Tech Stack



Use:



* React

* TypeScript

* Vite

* TailwindCSS

* shadcn/ui

* React Router

* React Hook Form

* Lucide Icons

* Framer Motion animations

* Recharts for charts

No Firebase.



No SQL.

# Important



This is **only a frontend prototype**.



There must be:



* No backend

* No authentication

* No database

* No API

* No Flask

* No SQL Server

* No TensorFlow



Instead, **simulate AI analysis**, mission assignments, clustering, dashboards, and report updates entirely with mock JSON data and local React state.



The final result should look like a polished commercial Smart City SaaS application suitable for demonstrating an AI-powered waste collection management platform to professors, municipalities, or potential investors.
